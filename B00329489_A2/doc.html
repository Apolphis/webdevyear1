Summary

When I created this website, I used Notepad++ as the editor due to personal preference.
To sample the audio I used Audacity (I also used it to edit the audio) and exported the samples as an mp3 file.
I closely refered to w3 schools for features like the CSS3 radial backgrounds.

Accessibility

The navigation was designed to be simplistic and easy to use.
The chosen colour schemes were implemented for the users eye comfort, making any text easy to read.
